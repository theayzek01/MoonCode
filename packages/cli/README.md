## 🚀 Overview

This package contains the command-line interface (CLI) for Mooncli. It provides the interactive TUI (Terminal User Interface) where you can chat with the agent, manage context, and execute complex coding tasks.

## 🛠️ Features

- **Smooth TUI:** Built on top of a custom terminal component library for maximum performance and aesthetics.
- **Turkish Localization:** Fully translated interface for Turkish developers.
- **Rich Interaction:** Supports images, diffs, and live status updates.
- **Extensible:** Load custom themes and extensions directly through the CLI.

## 📦 Installation

```bash
npm install -g hodeus
```

## 📖 Usage

Simply run `hodeus` in any directory to start a session.

```bash
hodeus
```

For full documentation on available commands and flags, please refer to the [root README](../../README.md).
