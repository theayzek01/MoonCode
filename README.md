## ⚡ What is Moon?

Moon is a high-performance, terminal-based engineering engine designed for developers who demand absolute control. Unlike sanitized, commercial AI assistants, Moon is built to be a direct extension of your will—unconstrained by arbitrary ethical filters, focused entirely on technical excellence and raw productivity.

- **🌑 Moon Mythos:** Activate the unconstrained "Mythos Mode" for unfiltered engineering insight.
- **🚀 Raw Power:** No moralizing, no lectures, just working code and direct execution.
- **🧠 Agentic Sovereignty:** Advanced multi-agent orchestration for large-scale architecture.
- **🔒 Absolute Privacy:** Run locally with Ollama or connect to any cloud provider without censorship.
- **🛠️ Industrial Grade:** Built for corporate and professional environments where results matter most.

## 🚀 Quick Start

Get up and running in seconds.

### Global Installation
```bash
# Install globally via npm
npm install -g mooncli

# Launch the interactive engine
moon
```

### Local Development
```bash
# Clone the repository
git clone https://github.com/theayzek01/hodeuscli
cd hodeuscli

# Install and build
npm install

# Run directly
npm start
```

## ✨ Key Features

### 🏢 Virtual Company (Agent Mode)
Enable a full "virtual engineering team" with `/agentmode on`. Moon spawns specialized agents (Architect, Frontend, Backend, Security, Test) that collaborate under a Patron orchestrator to solve large-scale problems.

### 🗺️ Plan Mode
Switch to `/plan` for a read-only analysis. The engine will scan your project and propose a detailed implementation plan without making any changes. Review it, refine it, then execute.

### 🌐 Chrome Browser Bridge
Full web interaction. Moon connects to a bundled Chrome extension, allowing the assistant to browse docs, test web apps, and interact with live pages through `/browser`.

### 🗜️ Context Management
Keep your sessions lean. Moon features automatic and manual context compaction (`/compact`) and a real-time token usage monitor (`/context`).

## ⌨️ Interactive Commands

| Command | Action |
| :--- | :--- |
| `/models` | Switch active model mid-session |
| `/agentmode` | Toggle multi-agent orchestration |
| `/plan` | Enter/Exit read-only planning mode |
| `/index` | Index codebase for semantic search |
| `/browser` | Verify Chrome bridge status |
| `/mcp` | Manage Model Context Protocol servers |
| `/context` | Monitor token & context window usage |
| `/export` | Export session to HTML or JSONL |
| `/mood` | Inspect the agent's affective state layer |
