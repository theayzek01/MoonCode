# MoonCode

Terminal tabanlı kod asistanı.

## Özellikler

- Hızlı komut satırı kullanımı
- Kalıcı oturum desteği
- Tarayıcı köprüsü
- MCP desteği
- TUI arayüzü

## Kurulum

```bash
npm install
npm run build
```

## Çalıştırma

```bash
mooncode
```

## Temel Komutlar

- `/help` yardım
- `/brain` bellek ve öneri görünümü
- `/autothink` otomatik düşünme
- `/browser` tarayıcı durumu
- `/mcp` MCP yönetimi
- `/doctor` sistem özeti

## Geliştirme

```bash
npm run check:ci
npm test
```

- Node.js 20+
