# MoonCode

Terminal odaklı, hızlı ve otonom bir AI mühendislik aracı. Gereksiz özelliklerden arındırıldı, sadece işini yapar.

---

### Hızlı Kurulum

```bash
git clone https://github.com/theayzek01/MoonCode.git
cd MoonCode

npm install
npm run build
npm install -g ./packages/cli

mooncode
```

---

### Nasıl Kullanılır?

1. **Başlat:** Terminale `mooncode` yaz ve gir.
2. **Sor:** Yapmak istediğin işi doğal dilde söyle (Örn: "Auth sistemindeki hatayı bul ve düzelt").
3. **İzle:** MoonCode projeni tarar, çözüm üretir ve onayını bekler.
4. **Onayla:** Çözüm mantıklıysa terminalden onay ver, MoonCode dosyalara uygulasın.

---

### Öne Çıkanlar

- **Self-Correction:** Kod hatalarını otonom tarar ve testlerden geçene kadar düzeltir.
- **Swarm:** Karmaşık işleri paralel çalışan farklı ajanlara (mimar, coder vb.) böler.
- **Context Management:** VectorDB kullanarak sadece gerekli kodu LLM'e gönderir, token tasarrufu sağlar.
- **Browser Bridge:** Chrome eklentisiyle web araştırması ve otomasyon yapar.

---

### Komutlar

- `/swarm` - Çoklu ajan modunu başlatır.
- `/fix` - Hata iyileştiriciyi tetikler.
- `/evolve` - Sistemin kendini geliştirmesini sağlar.
- `/browser` - Tarayıcı bağlantısını yönetir.
- `/index` - Repo indekslemesi yapar.
